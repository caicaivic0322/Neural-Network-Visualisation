MNIST数字分类 - 神经网络推理可视化

神经网络交互式可视化

这个应用程序展示了一个在MNIST上训练的紧凑型多层感知器(MLP)。绘制一个数字，观察激活值如何在全连接层中实时传播。

工作原理：

  • 绘制： 在2D网格（左上角）上点击并拖拽来绘制数字
  • 观察： 观察您的绘画如何通过网络层在3D空间中传播
  • 预测： 查看图表中每个数字（0-9）的概率（右上角）

网络架构（标准导出）：

  • 输入层： 28×28像素网格（您的绘画）
  • 全连接层1： 784 → 64神经元，使用ReLU激活
  • 全连接层2： 64 → 32神经元，使用ReLU激活
  • 输出层： 32 → 10个logits → Softmax概率

3D控制：

  • 旋转： 按住左键并拖拽
  • 平移： 按住右键并拖拽
  • 缩放： 使用鼠标滚轮

颜色编码：

  • 节点： 颜色表示激活强度（深蓝色表示低/负值，鲜艳的珊瑚色表示强正激活）
  • 连接： 暖色表示强正贡献，冷色表示负影响，浅色表示接近零。

训练自己的模型：

  • 运行 python training/mlp_train.py 来训练MLP（包括Apple Metal加速，如果可用）。
  • 脚本会写入 exports/mlp_weights.json，可视化器在加载时读取此文件。
  • 通过 training/mlp_train.py 中记录的CLI选项来更改隐藏神经元数量、训练轮数或导出路径。

实时功能：

  • 层激活： 球体实例显示每个神经元的激活，带有颜色编码的强度。
  • 重要连接： 每个目标神经元突出显示其最强的输入权重，以提高可读性。
  • 实时概率： 条形图实时更新logits → softmax值。

网络被有意保持紧凑，以确保流畅的实时显示。您可以尝试使用其他层大小重新训练 - 只需保持架构精简，以保持3D视图响应迅速。
